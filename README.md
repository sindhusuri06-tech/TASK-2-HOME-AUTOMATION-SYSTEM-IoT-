# Task 2: Home Automation System (IoT)

## Objective
To develop a home automation system to control
multiple devices using IoT.

## Description
This project demonstrates controlling home devices
such as lights and fans using a mobile application.
Commands are sent wirelessly to the Arduino.

## Devices Controlled
- Light
- Fan

## Working
- Mobile app sends commands
- Arduino receives commands
- Devices turn ON/OFF accordingly

## Conclusion
This project represents a basic IoT-based home
automation system used in smart homes.
